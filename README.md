# 楚汉棋局 - 中国象棋人机对战

一款功能完整、规则严谨的中国象棋人机对弈应用，集成 Elephantfish.py AI 引擎。

## 功能特性

### 核心游戏功能
- **标准中国象棋规则**：完全遵循官方行棋规则
  - 棋子走法：将/帅、士、象/相、车、马、炮、兵/卒
  - 特殊规则：马蹩脚、象堵眼、炮打隔子、兵卒过河可左右移动
  - **将帅不能照面**：同一列中间无棋子时不能见面
  - **长将限制**：同一走法将军不能超过3次，该棋子将被禁止移动

### AI 对弈引擎
集成 Elephantfish.py 作为核心 AI 引擎：

| 难度 | 搜索深度 | 描述 |
|------|---------|------|
| 入门 | 3层 | 适合初学者练习 |
| 普通 | 5层 | 有一定战术判断 |
| 困难 | 7层 | 复杂战术组合 |
| 大师 | 8层 | 高难度挑战 |

### 游戏操作
- **新游戏**：清空棋局，重新开始
- **悔棋**：双方各走一步后可悔棋
- **难度切换**：对局中可随时切换难度
- **重新开始**：一键重置全局状态
- **音效开关**：🔊/🔇 按钮控制静音

### 音效系统
完整的音效体验（需用户点击页面后启动）：
- 🔊 拿棋音效：清脆的短音
- 🔊 下棋音效：木质撞击声
- 🔊 吃子音效：有力的打击声
- 🔊 将军音效：警告提示音
- 🔊 胜利音效：欢快旋律
- 🔊 失败音效：低沉声音
- 🔊 AI思考音效：轻微嗡嗡声
- 🎵 BGM：喜庆的中国风背景音乐（16秒循环）
- 🔊/🔇 音效开关：可随时静音/开启（不影响播放进度）

### 视觉反馈
- 选中棋子高亮显示
- 可移动位置标记提示
- AI走棋轨迹显示
- 吃子动画效果
- 实时回合状态显示
- 响应式布局，支持移动端

## 启动方式

### 方式一：双击启动
双击 `start.bat` 文件，自动启动后端和前端服务。

### 方式二：手动启动

**终端1 - 启动后端 AI 引擎：**
```bash
python server.py
```

**终端2 - 启动前端服务：**
```bash
python -m http.server 8080
```

然后访问 http://localhost:8080

## 安装依赖

首次运行前需要安装 Flask 和 flask-cors：
```bash
python -m pip install flask flask-cors
```

## 技术栈

- **前端**：HTML5 Canvas + JavaScript
- **后端**：Python Flask + flask-cors
- **AI 引擎**：Elephantfish.py
- **音效**：Web Audio API
- **跨域**：CORS 支持

## 浏览器兼容性

- Chrome / Edge / Firefox 最新版
- 支持桌面端和移动端
- 需要现代浏览器支持 Web Audio API
